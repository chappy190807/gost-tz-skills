# ГОСТ 2.105. Правила оформления текстовых документов

Общие требования к текстовым документам (ЕСКД). Ниже — краткая выжимка правил, применимых при оформлении ТЗ.

## Построение документа

- Текст документа разделяют на **разделы, подразделы, пункты и подпункты**.
- Разделы нумеруют **арабскими цифрами** без точки в конце номера, записывают с абзацного отступа: `1`, `2`, `3`.
- Подразделы нумеруют в пределах раздела: `1.1`, `1.2`; пункты — в пределах подраздела: `1.1.1`, `1.1.2`.
- Каждый раздел рекомендуется начинать с нового листа (страницы).
- Заголовки разделов и подразделов записывают с прописной буквы, без точки в конце, не подчёркивают. Переносы слов в заголовках не допускаются.

## Изложение текста

- Текст излагают **безличной формой** («система обеспечивает…», «предусматривается…», «следует применять…»), в настоящем времени.
- В требованиях применяют слова «должен», «следует», «необходимо», «требуется, чтобы».
- **Единство терминологии:** недопустимо применять для одного понятия разные научно-технические термины, близкие по смыслу (синонимы), а также применять один термин в разных значениях.
- Не допускается применять произвольные словообразования, сокращать обозначения единиц величин без числа, использовать иностранные термины при наличии равнозначных русских.
- Числовые значения величин указывают с обозначением единиц. Диапазоны — «от … до … включительно».

## Перечисления

- Перед каждым элементом перечисления ставят дефис или, при необходимости ссылки, строчную букву со скобкой: `а)`, `б)`, `в)`. Не используют буквы **ё, з, й, о, ч, ь, ы, ъ** (перечень из 8 букв закреплён редакцией ГОСТ Р 2.105-2019, п. 6.7.5 со ссылкой на п. 6.3.5). Для дальнейшей детализации — арабские цифры со скобкой: `1)`, `2)`.
- Каждый элемент перечисления записывают с абзацного отступа.

## Таблицы

- Таблицы нумеруют арабскими цифрами в пределах раздела или сквозной нумерацией. Над таблицей слева помещают надпись «Таблица N — Наименование».
- На все таблицы должны быть ссылки в тексте.

## Формулы и иллюстрации

- Формулы нумеруют в круглых скобках справа. Иллюстрации подписывают «Рисунок N — Наименование».

## Приложения

- Приложения оформляют как продолжение документа. Каждое начинают с новой страницы с надписью «Приложение» и его обозначением (буквой русского алфавита). Приложения могут быть обязательными, рекомендуемыми, справочными.

## Сокращения

- Применяют только общепринятые сокращения (по ГОСТ) либо вводят перечень принятых сокращений. При первом упоминании термина приводят полное наименование, а в скобках — сокращение или аббревиатуру, далее используют сокращение.

---

**Итог для ТЗ:** сквозная нумерация разделов арабскими цифрами; безличная форма и настоящее время; единый термин для каждого понятия; каждое требование — проверяемое; таблицы, перечисления и приложения оформлены по правилам выше.
